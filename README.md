### 🛡️ Mailedit
**Phishing Warning Web App & Chrome Extension**

Extracts URLs from user-submitted or visible email content, runs them through live threat lookups, and returns a threat score with a safe/phishing verdict. Ships with content scripts for **Gmail and Outlook** that read the visible email body, flag suspicious links, and surface an in-page warning — with the web dashboard and extension sharing one consistent risk-scoring flow.

`JavaScript` `React` `Node.js` `Chrome Extensions API` `Vercel`
